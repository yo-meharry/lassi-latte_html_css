# Lassi & Latte Lounge

A fully responsive cafe website built using only **HTML & CSS** - no JavaScript.

---

## Features

- **Sticky Navbar** - stays at top while scrolling, with search bar
- **Main Section** - background image with gradient overlay, stats
- **Sidebar Layout** - sticky sidebar with navigation, hours & today's special
- **Menu Table** - rowspan, price badges, hover effects
- **Photo Gallery** - CSS Grid mosaic layout with hover overlay animation
- **Reviews** - 3-column review cards with decorative quote
- **Responsive Design** - mobile-friendly via media queries

---

## Concepts Learned

### HTML
- Semantic tags - `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`
- `<details>` & `<summary>` 
- `id` vs `class` - and using `id` for in-page anchor navigation
- `<span>` vs `<p>` vs `<div>` - when to use which

### CSS
- **CSS Variables** - --gold, --bg, --surface etc.
- **Flexbox** - display: flex, align-items, justify-content, flex-direction, flex-wrap
- **CSS Grid** - grid-template-columns, 1fr, grid-row, grid-column
- **Position** - relative, absolute, sticky and how they interact
- **z-index** - controlling layer stacking order
- **Box Model** - margin vs padding vs border
- **Display values** - inline, block, inline-block
- **Pseudo-elements** - ::before, ::after for adding content via CSS
- **Pseudo-classes** - :hover, :last-child, :visited, :active 
- **Transitions** - smooth hover animations
- **Transform** - scale(), rotate(), translateY(), transform-origin
- **object-fit: cover** - images that fill containers without distortion
- **linear-gradient** - overlays
- **calc()** - math inside CSS (height: calc(100vh - 64px))
- **overflow-y: auto** - scrollable sidebar
- **will-change** - GPU acceleration for smooth animations
- **Media Queries** - @media (max-width: 860px) for mobile layout
